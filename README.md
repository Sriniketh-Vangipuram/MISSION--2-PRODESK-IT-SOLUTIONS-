# 💰 Cash Flow Tracker

A fully functional Salary & Expense Tracker built using **Vanilla JavaScript**, focusing on **DOM manipulation, state management, and data persistence**.

---

## 🚀 Features

- 💵 Add and display total salary
- 🧾 Add expenses with name, amount, and category
- 📊 Real-time balance calculation
- 🗑 Delete expenses instantly
- 💾 Data persistence using LocalStorage
- 📈 Pie chart visualization using Chart.js
- 🎨 Multi-category expense tracking
- ⚠️ Smart validations (no negative/empty input, no overspending)
- 💬 Custom modal popup (no alert usage)
- 🎯 Responsive UI

---

## 🧠 Tech Stack

- HTML5
- CSS3 (Modern UI + Responsive Design)
- JavaScript (DOM + Logic)
- Chart.js (Visualization)
- LocalStorage (Persistence)
